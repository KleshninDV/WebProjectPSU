**HelpDesk System (Система управления заявками)**
Проект для итоговой работе по дисциплине "Язык программирования Java"

Проект представляет собой Spring Boot REST API приложение с веб-интерфейсом (JS + HTML) для управления системой технической поддержки. Приложение использует базу данных PostgreSQL и содержит 3 связанные таблицы.

**Функциональность**

Реализован полный цикл CRUD (Create, Read, Update, Delete) для всех сущностей:
**Просмотр данных:** Отображение списков пользователей, заявок и комментариев в табличном виде.
-**Добавление данных:**
  -Регистрация пользователей.
  -Создание заявок (с привязкой к пользователю).
  -Добавление комментариев к заявкам.
-**Редактирование:** Изменение заголовков, описания и статуса заявок (OPEN/CLOSED).
-**Удаление:** Удаление записей с поддержкой каскадного удаления (при удалении пользователя удаляются его заявки).
-**Валидация:** Проверка вводимых данных (корректность email, обязательные поля).
